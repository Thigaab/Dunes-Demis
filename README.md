# Dunes & Demis - Landing Page 4L Trophy

Landing page moderne pour le projet 4L Trophy **Dunes & Demis**.

## Contenu de la page

- Hero section avec identite adventure (sable + pastels)
- Section Mission (humanitaire et sportif)
- Timeline de preparation
- Footer avec liens reseaux sociaux
- Iconographie basee sur `lucide-react`

## Stack

- Next.js (App Router)
- React
- Tailwind CSS v4
- `next/font` (Geist + Oswald)
- `lucide-react`

## Demarrage rapide

```bash
npm install
npm run dev
```

Ouvrir ensuite `http://localhost:3000`.

## Scripts utiles

```bash
npm run lint
npm run build
npm run start
```

## Fichiers principaux

- `src/app/page.tsx`: structure de la landing page
- `src/app/globals.css`: theme global adventure (palette et utilitaires)
- `src/app/layout.tsx`: metadata, langue (`fr`) et fontes
